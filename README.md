# Aurora Siger — Relatório de Pré-Decolagem

## Sobre o projeto
Sistema de verificação automatizada para autorizar ou abortar o lançamento 
da nave Aurora Siger, desenvolvido como atividade integradora do curso de 
Ciência da Computação.

## O que o código faz
- Lê os dados de telemetria da nave
- Verifica se cada parâmetro está dentro da faixa segura
- Calcula a autonomia energética
- Emite o resultado: PRONTO PARA DECOLAR ou DECOLAGEM ABORTADA

## Como executar
1. Acesse o arquivo aurora_siger.ipynb neste repositório
2. Clique em "Open in Colab"
3. Clique em "Executar tudo"
